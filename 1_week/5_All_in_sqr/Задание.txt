Реализуйте шаблонную функцию Sqr, которая работает не только для чисел, 
но и для контейнеров. Функция должна возвращать копию исходного 
контейнера, модифицировав его следующим образом:

для vector элементы нужно возвести в квадрат;
для map в квадрат нужно возвести только значения, но не ключи;
для pair в квадрат нужно возвести каждый элемент пары.

Функция должна корректно работать не только для контейнеров, 
состоящих из чисел, но и для составных объектов, например, 
векторов словарей пар чисел.
